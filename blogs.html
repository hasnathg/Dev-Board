Question-1: What are the different ways to select an element in the DOM?

The main ways to select elements in the DOM are: getElementById, querySelector, querySelectorAll, getElementsByClassName, getElementsByTagName. These methods help us to target specific
HTML elements within our webpage's structure. Essentially, they are like search tools for the HTML.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content within an element, including HTML tags. innerText gets or sets the visible text content, ignoring HTML tags and respecting rendered styles. textContent
gets or sets all text content, including hidden text, without interpreting HTML.

Question-3: What is event delegation in the DOM?

Event delegation involves attaching a single event listener to a parent element, rather than individual child elements. When an event occurs on a child, it "bubbles up" to the parent, where the
listener can handle it based on the target. This improves performance by reducing the number of event listeners.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event on a child element propagates up to its parent elements. If a click happens on a button inside a div, the div's click event also triggers. It's like
ripples in water, starting from the source and spreading outwards.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create elements with document.createElement('div'), add them with appendChild(), and remove them with removeChild(). These functions allow to dynamically modify the HTML
structure of web page.
